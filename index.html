About Me

My name is Molly Zurheide. I’m a Portsmouth, New Hampshire-based front-end web developer with a wide-ranging educational and occupational background. To me, technology is exhilarating, and I love being part of a field that is revolutionizing the way we live. As an adolescent, I taught myself HTML and CSS because I loved using my artistic sensibilities to create content. Though I don’t come from a coding background, I possess skills and traits that make me well suited for web development. I am a top-notch problem solver, persistent, meticulously organized, creative, and easy to work with. My portfolio demonstrates my front-end expertise and my eye for attractive and intuitive design. My primary interests as a developer include optimizing interface to improve the user experience and front-end work with jQuery and React.
